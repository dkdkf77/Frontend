**VS Code 단축키 !**

사이드 바 열고 / 닫기 

cmd + B

빠른 열기(파일이나 기호 탐색)

cmd + p

모든 명령 표시(에디터의 모든 명령에 접근)

cmd + shift + p

현재 탭 닫기 

cmd + w

찾기 

cmd + f

→ 바꾸기

cmd + opt + f

하나의 tag 줄 위,아래로 이동

opt + up, down

하나의 tag 줄 위,아래로 복사 

alt + shift + up, down

들여쓰기 

tap → space 4번 한것 

공백 2를 권장 !

내어쓰기 

shift + tap
